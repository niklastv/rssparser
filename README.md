A simple Python script that fetches the latest news headlines from news sources (Yle, HS, BBC, Der Spiegel) directly to your terminal.

## Features
* Fetches news from multiple RSS feeds (configurable)
* Displays the 5 most recent headlines per source.
* Easily extensible with new news sources.

## Installation
1. Clone the repository: `git clone https://github.com/niklastv/rssparser.git`
2. Install dependencies: `pip install -r requirements.txt`
3. Run the application: `python rssfeedparser.py`
